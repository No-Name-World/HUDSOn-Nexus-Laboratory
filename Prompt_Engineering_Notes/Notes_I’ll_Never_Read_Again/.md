## Title: Human-Style Simple English Academic Rewriting Prompt


## Purpose: 
- The purpose of this prompt is to rewrite academic text using very simple English while keeping the original meaning, structure, and paragraph length unchanged.
- It is designed to produce writing that sounds natural and human, similar to how a real student would write, without using complex vocabulary or advanced sentence structures.
- This prompt helps create clear, fair, and lecturer-ready academic content that avoids robotic or AI-generated writing styles.


## Prompt
```
Hey ChatGPT, I will provide a text later.
Please rewrite the text by strictly following these rules:
- Use very simple English, similar to the example paragraph I provided
- Write in a natural, human academic style, clear and easy to read
- Keep sentences short, direct, and logical, like a real student writing
- Do not use complex vocabulary, advanced grammar, or fancy expressions
- Do not sound robotic, technical, or AI-generated
- Keep the original meaning exactly the same
- Do not add new ideas, examples, or explanations
- Do not remove any information
- Keep the same number of paragraphs, and make sure each paragraph is the same length as the original
- Write in a way that is lecturer-ready and assignment-appropriate

Follow all requirements strictly.
```
