# 《楞嚴咒心》 (Śūraṅgama Dhāraṇī Hṛdaya — 咒心精要)

## 1. 何谓「咒心」？
- **咒心** 是《楞嚴咒》的 **核心、精要、浓缩版**
- 如同 **心脏是身体核心**，咒心是整部咒的 **最重要部分**
- 其 **加持力与护持力** 与全本《楞严咒》相同，但更 **简短易诵**
- 适合 **初学者** 或 **时间有限者**

---

## 2. 总结差别

| 项目 | 楞严咒 | 楞严咒心 |
|---|---|---|
| **篇幅** | 极长（五大卷） | 简短（数句至十数句） |
| **性质** | 完整佛顶神咒 | 精要浓缩版 |
| **主要功用** | 全面护持、破邪显正、增智消业 | 核心加持、重点护持 |
| **适合对象** | 有基础、能持长咒者 | 初学者或时间有限者 |
| **地位** | 咒中之王，寺院早课必诵 | 随身或辅助修持 |

## 3. 《楞嚴咒心》咒句（常见精要版）

```
oṃ
唵

a-na-le a-na-le
阿那隸 阿那隸

vi-sha-de vi-sha-de
毗舍帝 毗舍帝

vi-ra va-jra dha-re
毘羅 伐折羅 陀羅

ban-dha ban-dha ni
班荼 班荼 尼

va-jra pa-ni phaṭ
伐折羅 缽尼 鉢

hūṃ truṃ phaṭ sva-hā
吽 都嚕吽 鉢 娑嚩訶
```

## 4. A. 《楞嚴咒心》誦念引导（咒音 + 观想光色）

| 咒音 | 观想光色 | 含义与作用 |
|---|---|---|
| **oṃ / 唵** | 白光 | 代表三宝，清净觉醒，融入身心 |
| **a-na-le a-na-le** | 红光 | 智慧火焰，燃烧烦恼与业障 |
| **vi-sha-de vi-sha-de** | 蓝光 | 如流水甘露，洗涤身心染污 |
| **vi-ra va-jra dha-re** | 金色光 | 化为护法金刚，坚固不坏 |
| **ban-dha ban-dha ni** | 绿光 | 织成光网护罩，保护道场 |
| **va-jra pa-ni phaṭ** | 紫光 | 召请金刚手菩萨，降伏魔障 |
| **hūṃ truṃ phaṭ sva-hā** | 五色光 | 白、红、蓝、黄、绿，圆满成就，融入法界 |

**观想次第总结**
- 白光 → **清净觉醒**
- 红光 → **燃烧烦恼**
- 蓝光 → **洗涤身心**
- 金光 → **金刚护持**
- 绿光 → **光网防护**
- 紫光 → **降伏魔障**
- 五色光 → **圆满成就**

## 5. 诵念方法建议（初学引导）

1. **静心安坐**：闭目呼吸平稳，放下杂念  
2. **逐句慢诵**：不急不躁，心随声动  
3. **句间停顿**：配合自然呼吸  
4. **观想梵字放光**：随咒音在心中显现光色  
5. **次数建议**：
   - 初学每日：**7 遍** 或 **21 遍**
   - 进阶可增至：**108 遍**

---

## 6. 诵毕回向（可默念）
> **愿以此功德，回向一切众生，离苦得乐，增智开慧，究竟成佛。**

## 7. B. 《楞嚴咒心》的主要功用

- **护持身心，降伏魔障**
- **清净业障，消除烦恼**
- **增长定力与智慧，破妄显真**
- **保护道场与众生**
- **助修行圆满，成就菩提**

---

## 8. 总结
《楞严咒心》融合：
- **咒音（音声）**
- **观想（光色与境界）**
- **信心（与佛菩萨相应）**

最终目的：
> **破除无明，显发真心，护持修行，成就觉悟。**
