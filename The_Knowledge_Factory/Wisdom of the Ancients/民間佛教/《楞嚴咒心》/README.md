# 《楞嚴咒心》 (Śūraṅgama Dhāraṇī Hṛdaya — 咒心精要)

## 1. 何谓「咒心」？
- **咒心** 是《楞嚴咒》的 **核心、精要、浓缩版**
- 如同 **心脏是身体核心**，咒心是整部咒的 **最重要部分**
- 其 **加持力与护持力** 与全本《楞严咒》相同，但更 **简短易诵**
- 适合 **初学者** 或 **时间有限者**

---

## 2. 总结差别

| 项目 | 楞严咒 | 楞严咒心 |
|---|---|---|
| **篇幅** | 极长（五大卷） | 简短（数句至十数句） |
| **性质** | 完整佛顶神咒 | 精要浓缩版 |
| **主要功用** | 全面护持、破邪显正、增智消业 | 核心加持、重点护持 |
| **适合对象** | 有基础、能持长咒者 | 初学者或时间有限者 |
| **地位** | 咒中之王，寺院早课必诵 | 随身或辅助修持 |
