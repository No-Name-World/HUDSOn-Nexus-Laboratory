# 千门八将108局 介绍

**千门八将108局** 是一种源自江湖谋略文化的博弈布局体系总集，强调心理攻防、信息结构、策略节奏与角色协作。108局象征 108 种完整的策略形态，覆盖设局、破局与控局的多种情境智慧。

体系核心为 **千门八将**（八种职能角色），并分为 **上八将（天局上策方向）** 与 **下八将（地局下策方向）**，分别对应高阶策略控局与基础环境心理入局。

---

## 🔺 上八将（36天局 · 上策 · 策略控局方向）
> 代表高阶布局、破局与节奏掌控（上策）

| 上八将 | 主要职能描述 |
|-------|-------------|
| 正将 | 全局策略统领者，决定布局主线方向与逻辑 |
| 反将 | 认知牵制与逆向思维设计，测试并引导目标判断偏向布局方向 |
| 提将 | 关键情报提取者，寻找推动局面的核心触点 |
| 谣将 | 认知影响与信息传播框架构建，塑造目标对局面的理解结构 |
| 除将 | 清除干扰与障碍，确保布局节奏顺畅不被阻断 |
| 风将 | 造势与心理环境影响，使情绪与氛围配合布局发展 |
| 火将 | 推动关键行动爆发点，执行策略突破节点 |
| 脱将 | 破局后安全退出或策略切换设计，确保可退、可换、可控 |

### 上八将特点：
- 先定策略，再定行动
- 关注信息结构与节奏掌控
- 适合破局、反制与控盘

---

## 🔻 下八将（72地局 · 下策 · 环境心理入局基础）
> 代表基础设局、现实借势与心理路径构建（下策）

| 下八将 | 主要职能描述 |
|-------|-------------|
| 正将 | 局面执行主线推进者，依赖环境条件入局 |
| 反将 | 制造认知偏差或心理对立，常从目标弱点切入操作 |
| 提将 | 锁定目标心理漏洞或现实需求作为入局切点 |
| 谣将 | 扩散现实信息诱导目标，而非宏观认知框架 |
| 除将 | 处理现实阻力与人际/环境障碍 |
| 风将 | 现实心理氛围铺垫与情绪引导 |
| 火将 | 执行关键行动，但策略深度较浅，执行重于战略 |
| 脱将 | 设计后路，但偏保守，用于拆局退场而非策略切换 |

### 下八将特点：
- 依赖现实环境、人性需求与心理破绽
- 先入局，再随机调整
- 技巧重于策略，是上八将运作的底层基础

---

## 🧠 上八将 vs 下八将 关键差异

| 维度 | 上八将（36天局上策） | 下八将（72地局下策） |
|------|-------------------|-------------------|
| 关注重点 | 全局策略与节奏控局 | 现实环境与心理路径入局 |
| 思路方向 | 破局、控盘、反制 | 设局、诱导、铺垫 |
| 信息结构 | 影响认知框架 | 影响现实信息 |
| 决策方式 | 先策略 → 后行动 | 先入局 → 后调整 |
| 体系定位 | 高阶控局（上策） | 基础入局（下策） |

---

## 108局体系层级定位

| 体系 | 层级 | 作用方向 |
|------|------|--------|
| 72地局 | 基础下策 | 设局与心理路径构建 |
| 36天局 | 高阶上策 | 破局与全局控盘 |
| 108局 | 八将协作总集 | 108种完整策略形态 |

---

## 📌 一句总结

**上八将负责策略控局（36天局上策），下八将负责环境心理入局（72地局下策），108局是八将协作在不同博弈情境下形成的108种完整布局智慧总集。**

---

> *千门非术，而是局；局中博弈，博的是人性、信息与节奏。*
