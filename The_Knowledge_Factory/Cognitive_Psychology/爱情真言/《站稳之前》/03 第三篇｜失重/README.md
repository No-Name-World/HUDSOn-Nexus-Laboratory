# 第三章｜失重

松手之后的世界，第一件来敲门的不是自由，  
而是——**空白**。

没有吵闹的悲伤，也没有失控的焦虑。  
只是一种突然失去重量的漂浮。

她照常上学、上班、生活。  
背包里放着水壶、笔记本、充电线，  
日程表照旧被课程、会议、作业塞满。

但节奏像是别人的，  
情绪像是静音的，  
生活像是没有主题音乐的电影。

她不再等回应，  
却也不知道自己在等什么。

她曾怀疑自己是不是退步了。  
但真正的事实是：  
她不是在坠落，她是在**支点被拔起后，学习重建自己的重心。**

以前的情绪有明确锚点，  
现在锚点消失了，但新的重心还没完全长好。

她第一次意识到：

**失重不是失败，它只是一个人真正开始学会独立前的过渡。**
