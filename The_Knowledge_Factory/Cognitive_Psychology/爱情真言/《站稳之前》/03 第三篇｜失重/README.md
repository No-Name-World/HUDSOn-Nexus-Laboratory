## 第三篇｜失重

她是在某个早晨意识到“失重”的。

不是因为发生了什么事。  
恰恰相反——什么都没有发生。

手机闹钟响起，她按掉。  
窗外的天是灰白色的，像一张还没开始写的纸。

她躺了一会儿，没有立刻起身。  
不是累，也不是不想上班，  
只是突然不知道该用什么理由开始这一天。

她已经很久没有再抓住任何人了。

没有反复点开聊天窗口，  
没有等一句回复来决定情绪，  
也没有在沉默里反复怀疑自己。

按理说，她应该轻松才对。

可现实是——  
她感到一种奇怪的漂浮感。

像是站在地面上，却感觉不到重量。

她的生活看起来很正常。

照常上班，  
照常吃饭，  
照常在晚上洗完澡，把头发吹干。

可所有动作都像是按流程完成的。

没有焦虑，  
也没有期待。

她发现自己很少再情绪波动，  
可也很少真正感到快乐。

那是一种平稳，却空旷的状态。

她曾经怀疑过：  
是不是自己哪里出问题了。

毕竟以前的她，不是这样的。  
以前的她，会因为一条消息心跳加速，  
会因为一个眼神反复回味。

而现在，什么都没有。

她站在便利店的冷柜前，看着一排饮料，  
却迟迟没有伸手。

不是不知道选哪一个，  
而是不知道“想不想”。

心理学上，这个阶段常常被误解。

人们会说：  
“你是不是变冷了？”  
“你是不是还没走出来？”  
“是不是你其实很孤单？”

可她知道，不是。

她没有回头，  
也没有怀念。

她只是第一次，在没有任何外在支撑的情况下，  
直面自己。

而这个自己，还没有完全成形。

失重的感觉，并不可怕。  
真正让人不安的，是它没有参照物。

过去，她的重心在别人身上。  
现在，那根线被剪断了。

可新的重心，还没长出来。

她开始明白，  
为什么有些人会在这个阶段选择退回去。

回到熟悉的依附里，  
哪怕不舒服，  
至少确定。

而站在这里，需要耐心。

她开始学着不急。

不急着填满时间，  
不急着让生活看起来“有意义”。

她允许自己发呆，  
允许自己不知道答案，  
允许自己只是坐在窗边，看天色慢慢暗下来。

有些夜晚，她还是会感到空。  
但那种空，不再让她恐慌。

她知道，那不是缺失，  
而是空间。

有一天，她在下班路上突然意识到——  
自己已经很久没有问过：

“别人会怎么看我？”

这个发现让她愣了一下。

她站在人行道上，  
红灯亮着，  
车流从面前呼啸而过。

那一刻，她感到了一点点重量。

很轻，但真实。

失重，并不是坠落。  
它只是失去了旧的支点。

而新的支点，  
正在悄悄生成。
