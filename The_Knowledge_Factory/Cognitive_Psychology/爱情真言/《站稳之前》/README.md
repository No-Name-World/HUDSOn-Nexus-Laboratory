# 《站稳之前》

### 一个关于情感依附、失重过渡与自我站立的心理故事小册

---

## 📌 作品定位

这是一本以**现实生活为背景的心理成长故事合集**，  
用第三人称叙述，记录人在亲密关系与自我价值之间的挣扎、觉醒与重建。

它不是安慰剂，也不是方法论清单，  
而是**让读者在故事里看见自己，在经历里理解自己**的文字路径。

---

## 🎯 这本小册想陪你走的方向

- 不再用他人回应来决定自己的情绪  
- 不再靠被需要来证明自己的存在价值  
- 在关系之外，建立属于自己的重心  
- 让亲密变成连接，而不是考核或负担  

一句话概括核心主旨：

> **真正的稳定，不是抓住谁，而是你自己站得住。**

---

## 📘 阅读顺序建议

按顺序阅读，会更完整地看到心理变化的轨迹：

1. **放大器** — 关系像镜子，只会放大你原本的状态  
2. **捷径** — 依附看似轻松，却让人停在原地  
3. **失重** — 失去支点后，生活出现空白与漂浮  
4. **站稳** — 界限与选择权在日常里悄悄形成  
5. **并肩** — 成熟关系是同行，不是依赖  

每一章都是**生活化的心理故事 + 内在逻辑推进**，  
没有批评，也不贴标签，  
只是把那段“很多人都会走、却很少被写清楚的路”呈现出来。
