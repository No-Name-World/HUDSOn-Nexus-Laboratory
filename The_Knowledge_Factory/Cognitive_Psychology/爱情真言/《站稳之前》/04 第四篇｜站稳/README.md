## 第四篇｜站稳

她第一次意识到自己“站稳了”，  
并不是在某个重要的时刻。

而是在一件很小的事上。

那天有人临时约她。  
语气随意，时间仓促，像是突然想起她的存在。

她看着那条消息，没有立刻回复。

不是忙，  
也不是故作冷淡。

她只是很自然地想了一下：  
**我想去吗？**

这个念头让她愣了一下。

以前的她，不太会问这个问题。

有人约，就答应；  
有人靠近，就配合。

不是不情愿，  
而是习惯了把“被需要”当成优先级。

可那天，她发现自己是真的不想。

于是她回了一句简单的拒绝。  
没有解释，也没有补偿。

发出去的那一刻，她下意识等了一下——  
等对方的反应。

但什么都没有发生。

世界没有崩塌，  
关系也没有瓦解。

她只是继续走在回家的路上。

她开始在生活里，  
一次又一次地感受到这种“脚踩在地上”的感觉。

比如，她不再勉强自己参与不喜欢的社交；  
不再为了维持关系而压缩休息时间；  
也不再急着在沉默中填补空白。

有些夜晚，她一个人吃饭，  
灯光不算明亮，  
食物很普通。

可她吃得很安静，也很踏实。

她慢慢意识到，  
站稳并不是“变强”。

而是不再摇晃。

不因为别人靠近就兴奋，  
也不因为别人离开就失衡。

她开始有界限。

不是刻意设立的，  
而是身体先一步替她做出了选择。

她会在疲惫的时候说“不”，  
在不舒服的时候停下来，  
在犹豫的时候允许自己不立刻决定。

她发现自己不再频繁回忆过去。

不是刻意放下，  
而是那些回忆，已经不再具有重量。

它们像是被妥善放进了抽屉里，  
存在，但不再随时被翻出来。

她偶尔会想起曾经那个，  
为了确认自己是否被爱而不断用力的自己。

但那种感觉，已经很遥远了。

变化最明显的，是关系。

她身边的人变少了，  
却变得更轻松。

没有试探，  
没有拉扯，  
也没有必须扮演的角色。

她不再急着被理解，  
也不急着证明自己。

有些关系，自然地走远；  
有些靠近，却不急不躁。

她第一次体会到——  
原来关系可以不用耗力。

有一天，她在整理房间时，  
发现自己丢掉了很多不再需要的东西。

旧衣服、用不到的小物件、  
还有一些曾经舍不得删的对话截图。

她看着垃圾袋，  
忽然意识到——  
自己已经很久没有害怕“失去”这件事了。

不是因为她不在乎，  
而是因为她知道：

即使什么都不抓住，  
她也站得住。

站稳，并不是终点。

它只是一个状态——  
一种你终于不再向外寻找支点的状态。

她站在那里，  
不张扬，也不急。

可她心里很清楚，  
这一次，自己不会再倒回去了。
