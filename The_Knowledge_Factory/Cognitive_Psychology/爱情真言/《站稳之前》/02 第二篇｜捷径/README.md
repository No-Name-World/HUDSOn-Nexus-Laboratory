## 第二篇｜捷径

她开始意识到自己在逃避，是在一个周末的下午。

阳光很好，窗帘拉开了一半，  
屋子里却安静得不像话。

她坐在沙发上，电视开着，却没在看。  
桌上的咖啡早就凉了。

她本来打算出门的。  
也许去逛一逛，或者随便找一家咖啡店坐着。  
可最后，她什么都没做。

她拿起手机，下意识地点开聊天界面。  
没有新消息。

那一刻，她忽然感到一种说不清的空。

不是寂寞，  
而是一种必须独自面对时间的无力感。

她这才发现，  
原来过去很长一段时间里，  
她的生活是被“有人”填满的。

下班后有人聊天，  
周末有人安排，  
情绪有人承接。

哪怕只是简单的几句对话，  
也足以让时间变得有重量。

而现在，这些都空出来了。

她突然明白，  
自己为什么会那么渴望亲密关系。

不是因为爱得多深，  
而是因为那样，她就不用一个人面对生活。

她并不是没试过靠自己。

试过把日程排满，  
试过培养新的兴趣，  
试过告诉自己“一个人也很好”。

可那些努力，在夜深的时候，  
总会变得很脆弱。

她会在刷牙的时候走神，  
在洗碗的时候发呆，  
在回到空房间的一瞬间，  
感到一种无处安放的疲惫。

于是她开始理解，  
为什么“有人陪”会那么诱人。

那是一条捷径。

只要有人在，  
她就不必回答那些问题——  
“我是谁？”  
“我想要什么样的生活？”  
“如果没有被需要，我还剩下什么？”

关系替她挡住了这些问题。

不是因为关系多完美，  
而是因为它足够吵，  
吵到可以盖过内心的声音。

她曾经告诉自己：  
先这样也没关系，  
等以后再慢慢想清楚。

可时间久了，她发现，  
自己走得很快，却始终没有往前。

她越来越依赖那份陪伴，  
却越来越不确定自己。

只要关系一出现裂痕，  
她的世界就会跟着晃动。

她开始害怕失去，  
不是失去那个人，  
而是失去那种“不用独自面对”的状态。

真正让她停下来的，不是某一次分开。

而是一个再普通不过的瞬间。

那天，她站在超市的货架前，  
看着满满一排商品，  
却突然不知道自己想买什么。

不是选择困难，  
而是一种更深的茫然。

她意识到，  
当她习惯了依附，  
连“自己要什么”这件事，  
都会变得模糊。

那一刻，她终于承认了一件事——  
她走的那条路，确实轻松。

可它带不走她真正想去的地方。

捷径不会让人迷路，  
却会让人停在原地。

而如果她再继续走下去，  
失去的就不只是关系，  
而是对自己的信任。

那天晚上，她没有做出什么改变。

没有突然变得强大，  
也没有立刻站稳。

她只是坐在床边，  
第一次没有打开任何聊天窗口。

房间很安静。  
安静得让人有点不安。

可她心里隐约知道——  
自己已经走到了一个必须转向的地方。
