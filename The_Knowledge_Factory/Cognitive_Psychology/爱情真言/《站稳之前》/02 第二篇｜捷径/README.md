# 第二章｜捷径

她并不是不懂成长很难。  
她只是选择了看起来更轻松的路。

捷径不一定危险，但一定诱人。

比如周末，她常常被临时约出去。  
有人问她“在哪”，她就会立刻把时间腾出来。  
不是因为特别想出门，而是因为**有人需要她**。

以前的夜晚，她习惯让聊天窗口保持亮着。  
房间安静的时候，她会感到必须找点声音来撑住自己。  
哪怕只是几句对话，也能让时间变得不那么空。

她以为自己是在靠近幸福。  
其实她是在靠近**不必面对自己的热闹**。

她害怕的不是失去一个人，  
而是失去那种**被外界暂时代替的重心**。

心理学上，这是一种太常见的误会：  
以为依附是连接，以为回应是价值，以为被需要是方向。

直到她在便利店的冷柜前停了很久，  
盯着饮料，却迟迟没有伸手。  
那一刻她才懂：

> **捷径不会让人迷路，它只会让人停在原地。**
