# 第二章｜捷径

她曾以为，只要有人陪着，就不用面对自己。  
一句回应，能让情绪稳定；  
一次沉默，却能让世界摇晃。

她需要的不是那个人，  
而是**不用独自回答“我是谁”的感觉**。

她走得很快，因为关系替她铺好了路。  
她不必决定周末去哪、晚餐吃什么、时间怎么安排，  
因为“有人”本身，就是她生活的节奏。

她以为自己害怕失去爱，  
其实她害怕的是——  
**失去那个被外界暂时代替的重心**。

心理学上，这是一种常见却隐蔽的阶段：  
用关系的热闹，回避自我定义的沉默。

她真正停下来的那天，不是分开，  
而是在超市的饮料柜前，  
她第一次发现自己不知道：  
**想不想，而不是选哪一个。**

捷径不会让人迷路，  
但会让人停在原地。

那天晚上，她没有突然改变，  
她只是第一次没有打开聊天窗口。  
房间安静，但她第一次意识到：

> **她不是在等消息，她是在等重心。**

而当她终于承认这一点时，  
她才真正开始走向自己的生活，而不是退回关系。

并不是变强，  
而是**开始站稳的前一步**。
