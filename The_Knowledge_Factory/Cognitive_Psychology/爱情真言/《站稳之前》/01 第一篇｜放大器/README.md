## 第一篇｜放大器

她第一次察觉到不对劲，是在一个很普通的夜晚。

手机放在桌上，没有亮。  
她已经看过很多次了。

不是没有事情可做，  
洗好的衣服还没收，  
外卖盒还放在水槽里，  
电视开着，却一句台词都没听进去。

她只是坐在那里，等。

她告诉自己，只是随手看看而已。  
可每一次拿起手机，心跳都会快一拍。

有消息，她就松一口气。  
没有，她就开始想：  
是不是自己哪里说错了话。

她会回想白天的聊天记录，  
一句一句往上翻，  
试图找出那个“让气氛变冷”的瞬间。

她并不觉得自己是在依赖。  
她只是想确认——  
确认自己还被放在心里。

那段时间，她的情绪变得很不稳定。

对方的一句话，  
可以让她一整天都轻快。

而一次迟迟没有的回应，  
却足以让她怀疑自己存在的意义。

她开始变得小心翼翼。  
说话前反复斟酌，  
情绪藏得很深，  
生怕多一点，就会把人推远。

她以为这叫珍惜。

直到她发现，自己越来越累。

有一天，她突然意识到一件事。

对方并没有真的对她做什么。  
没有伤害，也没有刻意冷落。

可她却一直在崩塌。

那一刻，她第一次产生了一个念头——  
这段关系，好像只是在放大她内心原本就存在的不安。

她并不是因为对方而不安，  
而是因为不安，  
才紧紧抓住了这段关系。

那天晚上，她没有再等消息。

不是因为释怀，  
而是因为一种说不上来的疲惫。

她把手机扣在桌上，  
走进浴室，  
水声盖过了所有通知的可能。

镜子里的她，看起来很安静。  
也很陌生。

她忽然意识到，  
如果她不自己修复那些裂缝，  
再多的爱，也只会变成负担。
