# 第一章｜放大器

她曾以为，爱是填补裂缝的材料。  
后来才懂，爱不是修补工具，它更像一面放大镜。

那段时间，她的生活被一台小小的手机定义。  
桌上堆着洗净却未叠的衣服，耳机散在键盘旁，窗外偶尔传来摩托车的引擎声。

她忙得很具体，心却忙得很抽象。

她会因为一句“我在”，感到世界有了重量；  
也会因为一句未读，感到自己像被遗忘在空气里。

她把所有的重心，都挂在对方的反应上。  
而对方其实什么都没做——没有伤害，也没有离开。

真正被放大的，是她内心本就存在的：  
**渴望被确认、渴望被放在心里、渴望自己是重要的。**

放大镜让细小变得巨大，让模糊变得清晰。  
而她第一次意识到这一点时，才惊觉——  
原来让她情绪震荡的不是关系，  
而是**她把自己放在了一个无法稳定的支点上。**
