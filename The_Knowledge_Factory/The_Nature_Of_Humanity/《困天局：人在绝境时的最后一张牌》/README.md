# 《困天局：人在绝境时的最后一张牌》

> 这是一个关于**逆境清醒、自我保护心理、识别操控机制与关系稳定管理**的 GitHub 仓库。  
> 不是教你主动作恶，而是让你提前理解**人性在压力与恐惧下如何失衡**，并学会在困局里保持清醒，不被局势吞噬。

---

## 作者信息

* **笔名**：不是我写，是手自己动的
* **定位**：心理成长故事创作者
* **风格**：用非常简单、好理解、可长期调用的方式，把复杂困局拆成心理规律与成长工具

---

## 仓库核心理念

我们不需要比谁聪明，也不需要操控别人，我们只需要：

✔ 识别人性弱点，而不是利用它  
✔ 在被动局势中保持判断，而不是被牵着走  
✔ 明白猜忌如何破坏关系，从而提前保护信任  
✔ 学会把困境经验复盘成未来能重复调用的心理模型  

---

## 你会在这里看到什么？

内容围绕以下心理成长方向展开：

* **困境中的清醒判断是第一保护层**
* **猜忌是关系中最隐形但最强的破坏力**
* **人不是被道理说服，而是被局势驱动**
* **真正的自我保护不是反击，而是提前识别风险**
* **把经历转成思维工具，才能长期受益**
