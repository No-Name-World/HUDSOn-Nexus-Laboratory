# 《困天局：人在绝境时的最后一张牌》

> 这是一个关于**在被动困局中保持清醒判断、保护自我边界、理解人性机制、并把经历复盘成未来可调用心理模型**的 GitHub 仓库。  
> 不是教你操控别人，而是教你**稳住自己、看清局势、留下出口、持续校准、长期调用**。

## 仓库真正想解决的问题

人遇困局时，最危险的不是局面，而是：

✘ 结论下太快  
✘ 事实没确认  
✘ 视角太低  
✘ 情绪抢方向盘  

我们不对抗人，我们对抗**信息差、认知差、结论差、情绪失控**。

---

## 仓库核心思维能力方向

这个仓库希望帮你建立 4 种长期可用心理能力：

1. **困境暂停模型** —— 压力出现时先暂停，而不是马上投降或马上冲动  
2. **事实确认模型** —— 事实比感觉更可靠，确认比猜测更有力量  
3. **视角拉高模型** —— 你把视角抬高一点，出口就多一点  
4. **长期校准模型** —— 思维要回看、要修正、要重复用，才能变成习惯

## 你会在这里看到什么？

* **关系里的边界如何保护**
* **信任如何落地，而不是悬在嘴上**
* **怀疑如何溯源，而不是误伤关系**
* **局势如何推动人，而你如何不被推动**
* **经验如何变成未来的心理工具，而不是心理伤口**
