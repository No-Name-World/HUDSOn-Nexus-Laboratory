# 《清醒但有温度》  

> 这里没有复杂算计，只有**清醒的观察 + 温柔的边界 + 真实的成长逻辑**。  
> 这是一个用心理故事讲人性、用思考方式避坑、用温度稳定前行的创作仓库。

---

## 🧠 这个仓库的核心价值  
我们不会直接学到别人脑袋里的想法，但我们**可以学会观察行为、读懂模式、保护自己不受困**。  
仓库里的内容会帮你：

- 建立**健康边界**（不失去温度，也不失去自己）
- 学会**识人逻辑**（不盲目信任，也不否定他人）
- 管理**情绪表达方式**（不失控、不压抑、有场景）
- 复盘**成长思维**（未来可重复调用，不靠人靠方法）

📌 一句话总结：  
**看得清世界的人，才有力气温柔地走更远。**

## 📂 内容结构说明  
每个文档都会包含 11 条人性真相，每条都用下面的方式呈现：

1. **核心观点（先讲成果与意义）**
2. **详细解释（简单、清楚、易懂）**
3. **成长逻辑小故事（真实但不夸张、不危险）**
4. **复盘方向（未来可调用的心理思考方式）**
