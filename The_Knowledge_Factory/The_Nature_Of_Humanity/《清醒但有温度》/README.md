# 《清醒但有温度》

> 这是一个关于**心理成长、人性观察、自我保护与稳定社交**的 GitHub 仓库。
> 不是教你变复杂，而是教你**用思维走得稳，用温度走得久**。

---

---

## 🌱 你会在这里看到什么？

仓库里的内容来自对生活和人际关系的真实观察，重点围绕：

* **如何在关系里保持自我，但不伤害别人**
* **如何相信别人，但不依赖别人**
* **如何表达情绪，但不被情绪带走**
* **如何在利益出现时，仍然保持公平与尊严**
* **如何把经验复盘成未来能用的思考方式**

---

## 🧠 仓库的真正目的

我们无法读心，但我们可以**读行为、读模式、读互动逻辑**。
这个仓库像一个**成长思维工具包**，帮你提前明白：

✔ 关系需要呼吸，不需要燃尽
✔ 信任需要验证，不需要盲目
✔ 情绪需要表达，不需要爆炸
✔ 交换是平衡，不是施舍
✔ 成长靠自己，不靠别人承诺
